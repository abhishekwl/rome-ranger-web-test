# Rome Ranger CRUD Web App

## Instructions
- Clone the Repo
- Run `yarn` to install the required packages
- Run `yarn start` to start the development server
- You can view the web app at [localhost:3000](http://localhost:3000)

## Structure

```console

|───public
|───src
    |--- App.js
    |--- index.js
    |───app
        |───assets
        │   ├───images
        │   └───lottie
        ├───components
        │   ├───FamilyMember
        │   └───MainAppBar
        ├───config
        └───pages
            └───Homepage
```
